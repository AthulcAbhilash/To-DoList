# To-Do List

## Overview
This project was built during my 12th standard as an improvement over my earlier personal productivity tools.
After creating a Notes application, I wanted something more structured, specifically for managing daily tasks, deadlines, and priorities.
This To-Do List focuses on usability, clarity, and task tracking, making it easier to stay organized throughout the day.

## Features
- Add and manage daily tasks
- Mark tasks as completed
- Visual priority indication based on due dates
- Clean and structured UI
- Easy task tracking and management

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Learned
- Managing state in UI (tasks list)
- Handling user interactions efficiently
- Designing a better user experience compared to previous projects
- Structuring components for clarity and usability
